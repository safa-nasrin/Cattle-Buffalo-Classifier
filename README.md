# Cattle-Buffalo-Classifier
An AI-powered web application that classifies 6 cattle breeds (Ayrshire, Gir, Holstein Friesian, Jersey, Sahiwal) and 4 buffalo breeds (Jaffrabadi, Mehsana, Murrah, Nili Ravi) using EfficientNetV2S deep learning models built with TensorFlow/Keras and deployed via dual Flask backends.
